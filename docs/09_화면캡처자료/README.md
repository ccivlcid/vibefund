# 09_화면캡처자료

화면 구성·플로우 검증용 캡처 및 프로토타입 스크린샷을 보관합니다.

## 폴더 용도

- **개발 전**: 와이어프레임·목업 스크린샷
- **개발 중**: 구현 화면 캡처 (회원가입, 프로젝트 등록, 펀딩, 결제, 댓글 등)
- **테스트**: 주요 시나리오별 캡처
- **배포 후**: 실제 서비스 화면 아카이브

## 보관 규칙 (권장)

- 파일명: `화면명_날짜.png` 또는 `플로우_단계_설명.png`
- 하위 폴더: `01_회원`, `02_프로젝트`, `03_펀딩`, `04_커뮤니티` 등 기능별 구분 가능

## 보완: 캡처 권장 목록 (MVP)

| 구분 | 캡처 대상 |
|------|-----------|
| 회원 | 로그인 화면, 회원가입(이메일/OAuth), 프로필 수정 |
| **유저 페이지** | 마이페이지 진입, 프로필 보기·수정, 내 프로젝트 목록, 관심/후원·내 댓글 |
| 프로젝트 | 목록, 상세(미리보기·진행률), 등록 단계별 |
| 펀딩 | 목표 설정, 리워드 안내, 진행률 표시 (결제 화면은 추후) |
| 커뮤니티 | 댓글 목록·작성, 업데이트 게시 |
| **관리자 페이지** | 대시보드, 회원 관리, 프로젝트 검수(승인/반려), 댓글·신고 관리 |
