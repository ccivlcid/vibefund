## 시드 데이터 (관리자·샘플)

개발/검증용 관리자 계정과 샘플 데이터는 **루트 `supabase/seed.sql`** 에서 제공합니다.

- **실행**: Supabase 대시보드 → SQL Editor → `supabase/seed.sql` 내용 붙여넣기 후 Run
- **계정**: 파일 상단 주석에 이메일·비밀번호·역할 표 정리 (관리자: admin@vibefund.local / Admin123!)
