# VibeFund 참고 링크 목록

## 1. 프로젝트 리포지터리

- **Git**: [https://github.com/ccivlcid/vibefund](https://github.com/ccivlcid/vibefund)
- 클론: `git clone https://github.com/ccivlcid/vibefund.git`
- GitHub·Vercel 등에서 위 URL을 리포지터리로 연결해 배포할 수 있습니다.

## 2. 경쟁·유사 서비스 (PRD §2 참고)

- Product Hunt: 제품 론칭·투표 (펀딩 기능 없음)
- 국내 크라우드펀딩 플랫폼: 물리 상품 중심 구조 비교용

## 3. 타겟 사용자 도구 (PRD §4)

- Webflow, Framer, Bubble, Notion 등 노코드/바이브코딩 도구

## 4. 기술 스택 공식 문서

- Next.js: https://nextjs.org/docs
- Supabase: https://supabase.com/docs (DB·Storage 등. 인증은 백엔드 API 사용으로 Supabase Auth 미사용)
- Vercel: https://vercel.com/docs (배포·환경 변수)
- (추후 결제 도입 시) Stripe: https://stripe.com/docs
- FastAPI: https://fastapi.tiangolo.com/
- Tailwind CSS: https://tailwindcss.com/docs

## 5. 추가 예정

- 노코드 창업 트렌드 리포트
- 펀딩·리워드 설계 사례
- PCI·결제 보안 가이드
