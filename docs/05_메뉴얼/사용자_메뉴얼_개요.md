# VibeFund 사용자 메뉴얼 개요

## 1. 제작자 (프로젝트 올리기)

- **회원가입**: 이메일 또는 Google 로그인
- **프로젝트 등록**: 제목, 한 줄 설명, 상세 설명, 서비스 URL(필수), 카테고리, 썸네일
- **미리보기**: URL 입력 시 자동 생성 (iframe·OG 이미지)
- **펀딩 설정**: 목표 금액, 마감일, 최소 후원액, 리워드
- **업데이트**: 프로젝트 진행 상황 게시, 댓글 답변

## 2. 서포터 (후원하기)

- **프로젝트 탐색**: 카테고리·정렬로 목록 조회
- **체험**: 프로젝트 상세에서 웹사이트 미리보기(iframe) 확인
- **후원(현재)**: 펀딩 목표·리워드 안내 확인 (실제 결제 기능 없음. 추후 제공 예정)
- **진행률 확인**: 목표 금액·마감일·진행률 표시
- **댓글**: 질문·응원 작성, 제작자 답변 확인

## 3. 유저 페이지 (마이페이지)

- **진입**: 로그인 후 유저 전용 메뉴(마이페이지 등)에서 접근. 본인만 조회 가능.
- **프로필**: 이름, 아바타, 소개 등 보기·수정
- **내 프로젝트**: 등록한 프로젝트 목록, 상태 확인, 수정/삭제 진입
- **관심·후원**: (현재) 관심 프로젝트 등. 결제 도입 후 내 후원 목록
- **내 댓글·알림**: 본인이 쓴 댓글 또는 알림 요약(구현 범위에 따라 제공)

## 보완: 주의사항·FAQ

### 주의사항

- **서비스 URL**: iframe에 넣을 수 있는 공개 URL만 등록 (일부 사이트는 X-Frame-Options로 차단될 수 있음)
- **결제**: 현재는 결제 기능이 없습니다. 추후 도입 시 실명·카드 정보는 결제 대행사만 처리하며 플랫폼에 저장하지 않습니다.

### FAQ (예시)

| 질문 | 답변 |
|------|------|
| 지금 후원(결제)할 수 있나요? | 현재는 펀딩 목표·진행률 표시만 제공하며, 결제 기능은 추후 도입 예정입니다. |
| (추후) 펀딩 실패 시 후원금은? | 정책에 따라 자동 환불 또는 수동 환불 절차 안내 |
| 프로젝트 수정은 마감 전에만? | 정책에 따라 마감 전만 허용 또는 제한적 수정 허용 |
| 리워드 변경 가능 여부? | 이미 후원이 있는 리워드는 변경 제한 권장 |
